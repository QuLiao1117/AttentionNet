# AttentionNet on CIFAR10

AttentionNet is an new image classification architecture designed by Liao Qu.

## Prerequisites
- Python 3.6+
- PyTorch 1.0+
- vit_pytorch 0.19.4

## Training
```
# Start training with: 
python main.py --lr=0.003

# You can manually resume the training with: 
python main.py --resume --lr=0.003
```
